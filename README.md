# Sales & Profit Performance Dashboard

## Overview
This project analyzes sales performance, cost structure, and profitability trends using Power BI.

The objective was to provide executive-level visibility into revenue drivers and margin behavior.

## Key Features
- Revenue, Cost, and Profit KPI tracking
- Year-over-Year comparison
- Regional and product performance analysis
- Margin sensitivity evaluation
- What-if scenario simulation

## Tools Used
- Power BI
- DAX
- Excel

## Business Value
The dashboard enables structured, interactive analysis to support data-driven business decisions.
